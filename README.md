# Trip-Advisor-Clone
🌍 TripAdvisor Clone Website

A responsive TripAdvisor-inspired website clone built using HTML and CSS.
This project focuses on recreating the UI layout and design of the popular travel platform TripAdvisor.

🚀 Features

Clean and modern UI

Navigation bar similar to TripAdvisor

Destination cards & reviews section

Hover effects and smooth layout

Fully responsive design (mobile & desktop)

🛠️ Technologies Used

HTML5

CSS3 (Flexbox & basic animations)

🎯 Purpose

This project was created for learning and practice, to improve frontend skills and understand real-world website layouts.

📌 Note

This is not an official TripAdvisor website.
It is built only for educational purposes.
